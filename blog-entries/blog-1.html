With it being the first week of my blog and fairly early in the set-up process of it, the topic of web hosting seemed quite relevant.

What in the World is Web Hosting?

Web hosting is a service which enables for files, including code and images, composing a website to be accessible to users across the internet. Web server(s) store all files of a website and are managed by a hosting provider.

How do I get Started with Web Hosting?

Web hosting is an essential step of the web development process . . . if you plan on sharing your website with the world. The first step is to register a domain name for the website, which will be permant address where users can find your site. Secondly, you will need to choose a hosting provider and plan. (Occassionally domain names will come in package deals with a hosting plan.)

In a world like today, free web hosting can be difficult to find, although there are several cheaper options that seem to be pretty good.

What are Some Web Hosting Services?

There are many! I will discuss the more major types of web hosting services:

Shared Web Hosting

Often known as the cheapest and "entry-level" hosting service, shared web hosting provides consumers a shared physical server with the same resources, including CPU, RAM, and disk space. As a result of many users using the same resources, a low traffic and bandwidth website is most desirable on a shared web hosting platform. Minimal technical experience is required in using this hosting service.

A fairly cheap and decent shared web hosting service, I've used in past years is Blue Host. Definitely would recommend if your looking for a more budget-friendly and bang for your buck hosting service.

Dedicated Server Hosting

Dedicated server hosting is where you are renting a physical server from a hosting company. A dedicated server is considered to be the highest level of server needed for a buisness with high traffic and is typically one of the more expensive options. Often times hiring a system administrator is hired to work out the technical details for a dedicated server.

Virtual Private Server (VPS) Hosting

Virtual private servers share a single physical server, but in a way act like multiple by dividing it into smaller virtual servers. VPS hosting is in between shared hosting and dedicated machine hosting.

Cloud Hosting

Cloud hosting is one of the biggest growing forms of hosting today. In fact, this blog was configured with a cloud hosting service, Digital Ocean. While dedicated server hosting provides a single server for hosting, cloud hosting spreads out data accross multiple machines. One of the main advantages of cloud hosting is it's "elastic scaling", allowing for more CPU cores, RAM, and other recources to fit web traffic.

Do I Need a Web Hosting Service to Show Off My Website?

You totally can host a website yourself and purely from your own home basement if you wanted! Although, there are a number of limitations when going this route, it's more technical, and can be quite pricey.

To completely host a website yourself, you would need to have an (expensive) server with a constantly-running power supply and high bandwidth internet. You would also need to ensure your Internet Service Provider (ISP) would allow for you to run a website like this — not all do. Furthermore, it would require a high technical knowledge for setting-up and maintaining it, including knowledge on setting up "WWW" server software.
